# Roman to Integer

## תיאור השאלה

Convert a Roman numeral to an integer.

## פיתרון

Sum each symbol while subtracting when a smaller value precedes a larger one.

```javascript
function romanToInt(s) {
  const values = {
    I: 1,
    V: 5,
    X: 10,
    L: 50,
    C: 100,
    D: 500,
    M: 1000,
  };
  let total = 0;
  for (let i = 0; i < s.length; i++) {
    const current = values[s[i]];
    const next = i + 1 < s.length ? values[s[i + 1]] : 0;
    total += current < next ? -current : current;
  }
  return total;
}
```

```python
def romanToInt(s: str) -> int:
    values = {
        "I": 1,
        "V": 5,
        "X": 10,
        "L": 50,
        "C": 100,
        "D": 500,
        "M": 1000,
    }
    total = 0
    for i, char in enumerate(s):
        current = values[char]
        next_value = values[s[i + 1]] if i + 1 < len(s) else 0
        total += -current if current < next_value else current
    return total
```

# Two Sum / סכום שניים

## תיאור השאלה

ניתן להחזיר את התשובה בכל סדר.

## פיתרון

```javascript
const twoSum = (array, goal) => {
    let indexes = [];

    for(let i = 0; i < array.length; i++){
       for(let j = i + 1; j < array.length; j++){
          if (array[i] + array[j] === goal) {
        indexes.push(i);
        indexes.push(j);
          }
       }
    }
    return indexes;
}
```

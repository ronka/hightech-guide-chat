# Container With Most Water

## תיאור השאלה

Given `n` non-negative integers `height` where each represents a vertical line at coordinate `i`, find two lines that together with the x-axis form a container that holds the most water.

## פיתרון

Move two pointers towards each other while tracking the best area, always discarding the shorter line.

```javascript
function maxArea(height) {
  let left = 0;
  let right = height.length - 1;
  let best = 0;
  while (left < right) {
    const current = Math.min(height[left], height[right]) * (right - left);
    best = Math.max(best, current);
    if (height[left] < height[right]) {
      left++;
    } else {
      right--;
    }
  }
  return best;
}
```

```python
from typing import List


def maxArea(height: List[int]) -> int:
    left, right = 0, len(height) - 1
    best = 0
    while left < right:
        current = min(height[left], height[right]) * (right - left)
        best = max(best, current)
        if height[left] < height[right]:
            left += 1
        else:
            right -= 1
    return best
```

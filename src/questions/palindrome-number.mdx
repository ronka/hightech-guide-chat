# Palindrome Number

## תיאור השאלה

Given an integer `x`, return `true` if `x` is a palindrome, and `false` otherwise.

## פיתרון

Reverse the digits numerically and compare with the original value.

```javascript
function isPalindrome(x) {
  if (x < 0) return false;
  let original = x;
  let reversed = 0;
  while (x > 0) {
    reversed = reversed * 10 + (x % 10);
    x = Math.floor(x / 10);
  }
  return reversed === original;
}
```

```python
def isPalindrome(x: int) -> bool:
    if x < 0:
        return False
    original = x
    reversed_num = 0
    while x > 0:
        reversed_num = reversed_num * 10 + x % 10
        x //= 10
    return reversed_num == original
```

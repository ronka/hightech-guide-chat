# Two Sum

## תיאור השאלה

Given an array of integers `nums` and an integer `target`, return indices of the two numbers such that they add up to `target`.

- Each input is guaranteed to have exactly one solution.
- You may not use the same element twice.
- You can return the answer in any order.

## פיתרון

```javascript
function twoSum(nums, target) {
  const valueToIndex = new Map();
  for (let i = 0; i < nums.length; i++) {
    const complement = target - nums[i];
    if (valueToIndex.has(complement)) {
      return [valueToIndex.get(complement), i];
    }
    valueToIndex.set(nums[i], i);
  }
  return [];
}
```

```python
from typing import List

def twoSum(nums: List[int], target: int) -> List[int]:
    value_to_index = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in value_to_index:
            return [value_to_index[complement], i]
        value_to_index[num] = i
    return []
```

## דוגמאות

**Example 1:**
```
Input: nums = [2,7,11,15], target = 9
Output: [0,1]
Explanation: nums[0] + nums[1] = 2 + 7 = 9
```

**Example 2:**
```
Input: nums = [3,2,4], target = 6
```
